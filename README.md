# AWS Cloud Projects

## About
This repository contains AWS cloud development projects completed for coursework and hands-on learning.

## Technologies Used
- AWS Lambda
- API Gateway
- DynamoDB
- IAM
- S3
- EC2
- JavaScript
- NodeJS

## Projects Included

### Baseball Stats API
Created an API using API Gateway, Lambda, and DynamoDB to retrieve and add baseball statistics.

Features:
- GET requests
- POST requests
- DynamoDB integration
- Lambda functions
- REST API deployment

### S3 Thumbnail Generator
Used AWS Lambda and S3 buckets to automatically generate image thumbnails after uploads.

### EC2 IIS Web Server
Created a Windows EC2 instance and hosted a website using IIS.

### IAM User Groups
Configured IAM users, groups, and permissions for AWS resource access.

## What I Learned
- Cloud resource management
- Serverless architecture
- API deployment
- AWS permissions and security
- DynamoDB integration
- Lambda event handling
